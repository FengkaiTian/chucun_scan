# 别问

S&P 500 每日股票扫描系统，基于技术指标自动评分，生成 HTML 报告并发布到网页。

**网站：** https://fengkaitian.github.io/chucun_scan/

---

## 评分体系（满分100）

| 维度 | 最高分 | 说明 |
|---|---|---|
| 趋势强度 | 30 | MA5 > MA10 > MA20 多头排列 |
| 乖离率 | 20 | 股价偏离MA20的距离，越近越好 |
| 成交量 | 15 | 放量上涨加分，缩量下跌减分 |
| MACD | 15 | 金叉/零轴上方加分，死叉减分 |
| RSI | 10 | 30-60区间加分，超买超卖减分 |
| 支撑位 | 10 | 股价在MA20上方加分，跌破减分 |

| 分数 | 信号 |
|---|---|
| ≥ 80 | 强烈买入 ⭐⭐⭐ |
| 65–79 | 买入 ⭐⭐ |
| 50–64 | 观望 |
| < 35 | 回避 |

---

## 文件说明

| 文件 | 说明 |
|---|---|
| `daily_scan.py` | 主扫描脚本，运行后生成报告并推送到网站 |
| `loop_scan.py` | 循环脚本，美股开盘时间内每小时自动运行 |
| `RSI_analysis.py` | 历史统计：RSI跌破30后各时间段胜率分析 |
| `magic_nine_history_3y_vol.py` | TD九转策略历史回测 |
| `index.html` | 最新报告（自动生成） |

---

## 使用方法

**手动运行一次：**
```
python daily_scan.py
```

**开启每小时自动扫描：**
```
python loop_scan.py
```
